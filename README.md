# FORRT CV: Re-usable model of open science CV

This second iteration of the FORRT CV was developed by [Emily Friedel](https://orcid.org/0009-0001-0917-5398) as part of her work on [Team Credit](https://forrt.org/about/teams/). 

Please re-use and credit us! 

## Compiling the CV

For compiling the CV locally, you need a LaTeX distribution installed, like:

- **Windows:** MiKTeX
- **Mac:** MacTeX
- **Linux:** TeX Live

Then, run in the terminal:

```
pdflatex FORRT_cv.tex
```
